Проекты:

Shelter

Приют - это сайт приюта для животных, где можно подробнее ознакомиться с информацией о каждом питомце, находящемся в приюте. Сайт разработан адаптивным дизайном и состоит из двух страниц, согласно макету в Figma.

Coffee house

Кофейня - это сайт, который предоставляет информацию о кафе и позволяет ознакомиться с его меню. Сайт разработан с адаптивным дизайном и состоит из двух страниц, согласно макету в Figma. (в разработке)

Nonograms

Японский кроссворд — игра головоломка, в которой зашифровано некоторое изображение, с помощью цифр. Целью головоломки является полное восстановление этого изображения. Есть возможность выбрать желаемый шаблон картинки для игры игра или случайный. Все необходимые элементы генерируются с помощью JavaScript.

Hangman

Виселица — классическая игра в слова, где нужно найти правильный ответ, угадывая буквы по одной. Можно использовать как виртуальную, так и физическую клавиатуру. Все необходимые элементы генерируются с помощью JavaScript. (в разработке)

Audio player

Аудиоплеер - это музыкальный плеер, позволяющий по очереди проигрывать музыкальные треки или перелистывать их с помощью кнопок. Каждому музыкальному треку соответствует определённое изображение, название и автор. Отображается продолжительность аудиотрека и его текущее время проигрывания.

Image Gallery

Галерея изображений - это приложение, которое отображает фотографии, полученные от API. В приложении реализован поиск: при вводе поискового запроса изменяются отображаемые фотографии.
